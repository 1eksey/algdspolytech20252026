# Лабораторная работа 26: Арифметика

## Условие задачи
Заданы два целых положительных числа A и B. Расставьте знаки арифметических операций (+, -, *, /) между цифрами числа A, чтобы получить B.

### Исходные данные
В текстовом файле input.txt записаны два числа A, B.

**Ограничения:**
- A ≤ 10^100
- B ≤ 2^32

### Результат
Выведите в текстовый файл output.txt арифметическое выражение, результатом которого является число B, или 0 если такого выражения построить нельзя.
